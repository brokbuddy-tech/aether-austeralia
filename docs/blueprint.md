# **App Name**: Aether Australia

## Core Features:

- AI-Powered Property Search & Discovery: A centralized hero search with BUY, RENT, and SOLD tabs. The system will leverage an AI tool to intelligently suggest 'Recent Searches' and 'Trending Areas' on focus to guide user exploration.
- Advanced Property Listing & Filtering: A dedicated property search page featuring a sticky top bar with comprehensive filters (Location, Price in AUD, Property Type, Bedrooms, 'Open for Inspection' times). Listing cards will display large-scale hero imagery with key details like Bed, Bath, Car, and Square Metres (m2) overlay.
- Detailed Property Presentation Page: Comprehensive property description page with an integrated image gallery, 360-degree virtual tour button, a detailed 'Property Story,' and an 'Upfront Costs' breakdown including Australian Stamp Duty estimates and a localized 'Verified Listing' badge (e.g., REIA compliant).
- Agent Contact & Interaction Hub: A sticky Agent Profile Sidebar on listing pages, featuring a circular headshot and direct 'SMS Agent' and 'WhatsApp' buttons for easy communication.
- Property Inspection Scheduler: On property description pages, users can schedule property inspections by selecting available 15-minute time slots.
- Localized Insights Centre: A blog-style grid displaying market updates and expert advice with '2 min read' tags, designed to emphasize local Australian knowledge and trends.
- Proprietor Portals for Reporting: Dedicated login blocks for Landlords and Vendors to view real-time income and expenditure reports for their properties.

## Style Guidelines:

- Primary action color: Royal Blue (#0047AB), derived from HSL(218, 100%, 34%), chosen for its corporate aesthetic and used for Call To Actions and key interactive elements.
- Background color: A very light, crisp cool gray (#F5F7FA), derived from HSL(218, 15%, 96%), maintaining a clean corporate aesthetic as a soft base. Pure 'Optical White' (#FFFFFF) is used for main content areas.
- Secondary accent color: A rich purple (#775BDD), derived from HSL(248, 70%, 60%), providing an analogous contrast to the primary Royal Blue. Additionally, 'Sandstone/Earth Tones' (#D4BF9A, from HSL 39, 32%, 72%) will be integrated specifically for 'Insights' sections, reflecting warm minimalism.
- Headline font: 'Inter' (sans-serif), with bold and extra bold weights for impactful titles. Body text font: 'Open Sans' (sans-serif) for high readability across the platform. Note: currently only Google Fonts are supported.
- Leverage high-key, natural light photography of Australian architecture for listings. Agent profiles will feature 'living-wall' and 'biophilic' backdrops. Implement circular icon cards for 'The McGrath Difference' section and a localized 'Verified Listing' badge for compliance.
- The homepage uses a 'pageless' section logic for fluid storytelling. Mobile navigation features a sticky 'Hamburger' menu opening a full-screen glassmorphic interface. Property search employs a sticky top filter bar. Listing cards prominently feature large-scale 'Hero' imagery with overlaid details. Property detail pages include a sticky agent profile sidebar.
- Desktop hover effects on property listing cards will trigger a subtle zoom-in on the hero image and dynamically reveal the agent's name for enhanced interactivity.
